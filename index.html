 TIPS → It will show you all databases, you can see your database named as database_1 in the databases list which is created just before by you

delete a database

DATA TYPES

For numeric fields

  • BIGINT (very big values)
  • INT (big values)
  • SMALLINT (-32,768 to 32,767)
  • TINYINT (0 to 255)
  • DECIMAL(m,d)
  • NUMERIC (same as decimal)
  • FLOAT(m,d)
NOTE : For float data type, display lenght (m) and number of decimals (d) are optional.

Date and time data types

  • DATETIME (YYYY-MM-DD HH:MM:SS)
  • DATE (YYYY-MM-DD)
  • TIME (HH:MM:SS)
  • YEAR (YYYY)

Character data types

  • CHAR(m)
  • VARCHAR(m)
    NOTE : VARCHAR(m) is best than CHAR(m) in terms of memory wastage.
  • TEXT (to store large amount of text)
For storing image
  • BLOB (binary large object)
Types of contraints
  • NOT NULL
  • DEFAULT
  • UNIQUE
  • PRIMARY KEY
  • FOREIGN KEY
  • CHECK CONSTRAINT
create table
CREATE TABLE supplier( SID int NOT NULL, NAME varchar(20) NOT NULL, ADDRESS varchar(80)); TIPS -> It will create a table named as supplier which has 3 attributes -
  • SID
  • NAME
  • ADDRESS
now type given below query - DESCRIBE supplier; you will see something like this. 
+---------+-------------+------+-----+---------+-------+
| Field   | Type        | Null | Key | Default | Extra |
+---------+-------------+------+-----+---------+-------+
| SID     | int(11)     | NO   |     | NULL    |       |
| NAME    | varchar(20) | NO   |     | NULL    |       |
| ADDRESS | varchar(80) | YES  |     | NULL    |       |
+---------+-------------+------+-----+---------+-------+
To insert values inside of table
There are two methods to insert values inside of table -

  • Method 1

    INSERT INTO supplier value( 1, "abhinav", "address_of_abhinav");

  • Method 2

    INSERT INTO supplier(SID, NAME, ADDRESS) value(1,"abhinav","address_of_abhinav");
To show table details

  • show full table detail

    SELECT * FROM supplier;

    you will get something like this -

    +-----+---------+---------------------+
| SID | NAME    | ADDRESS             |
+-----+---------+---------------------+
|   1 | abhinav | address_of_abhinav  |
+-----+---------+---------------------+

    Similary you can insert more values inside of table.

  • Show details of some attribute inside of a table

    SELECT SID, NAME FROM supplier;

    You will get something like this -

    +-----+---------+
| SID | NAME    |
+-----+---------+
|   1 | abhinav |
+-----+---------+

Logical operators
SELECT * FROM supplier WHERE SID=3 OR SID=1; TIPS → It will show information of the attribute SID whose SID=3 OR SID=1
SELECT * FROM supplier WHERE NAME LIKE 'c%'; TIPS → It will show information of the attribute NAME whose value is starting from c SELECT * FROM supplier WHERE SID LIKE '1_'; TIPS → It will show information of the attribute SID whose value is of length 2 which is starting from 1 and at the second character, value should be starting from 0 to 9 which means that it will show the details of those SID's whose value would be between 10 (included) and 19 (included) . SELECT * FROM supplier WHERE SID LIKE '1%'; TIPS → It will show information of the attribute SID whose value is of any lenght which is starting from 1 and ending at anything

Now you know the difference between '1_' and '1%'

SELECT * FROM supplier WHERE NAME IS NULL; TIPS → It will show information of the attribute NAME whose value is null (empty) SELECT * FROM supplier WHERE NAME IS NOT NULL; TIPS → It will show information of the attribute NAME whose value is not null SELECT * FROM supplier WHERE SID IN (1,2,5); TIPS → It will show information of the attribute SID whose value is 1 , 2 , 5.
If there is no value associating to the given SID then it will not show details of those SID's . SELECT * FROM supplier WHERE SID BETEEN 1 AND 10; TIPS → It will show information of the attribute SID whose value is between is 1 and 10
Update or change value inside of table
Suppose table named as supplier is look like this - 
+-----+---------+---------------------+
| SID | NAME    | ADDRESS             |
+-----+---------+---------------------+

+-----+---------+---------------------+

+-----+---------+---------------------+
|   1 | zoe     | address_of_abhinav  |
+-----+---------+---------------------+
|   2 | backus  | address_of_backus   |
+-----+---------+---------------------+
Delete a row from table
Suppose you want to delete a row from a table named as supplier whose SID = 2 (see just upper diagram). DELETE FROM supplier WHERE SID = 2;

Group by clause
SELECT NAME, COUNT(*) FROM supplier GROUP BY NAME; TIPS → It will count the frequency of each name in table supplier and display result.
Suppose table supplier looks like this - 
+-----+---------+---------------------+
| SID | NAME    | AGE                 |
+-----+---------+---------------------+
|   1 | abhinav | 12                  |
+-----+---------+---------------------+
|   2 | peterson| 15                  |
+-----+---------+---------------------+
|   3 | galvin  | 14                  |
+-----+---------+---------------------+
|   4 | lexi    | 20                  |
+-----+---------+---------------------+
|   5 | lexi    | 10                  |
+-----+---------+---------------------+
|   6 | abhinav | 85                  |
+-----+---------+---------------------+
When you run given below query - SELECT NAME, COUNT(*) FROM supplier GROUP BY NAME; then output will be - 
+---------+---------------------+
| NAME    | COUNT               |
+---------+---------------------+
| abhinav | 2                   |
+---------+---------------------+
| galvin  | 1                   |
+---------+---------------------+

Table result with changing the attribute name with 'AS' clause
SELECT NAME AS 'NAME_OF_SUPPLIER', ADDRESS AS 'ADDRESS_OF_SUPPLIER' FROM supplier; TIPS → It will show the table details with attribute NAME and ADDRESS as NAME_OF_SUPPLIER and ADDRESS_OF_SUPPLIER respectively.
Suppose table supplier look like this 
+-----+---------+---------------------+
| SID | NAME    | ADDRESS             |
+-----+---------+---------------------+
|   1 | zoe     | address_of_abhinav  |
+-----+---------+---------------------+
|   2 | backus  | address_of_backus   |
+-----+---------+---------------------+
OR you can give below query to get table details of supplier - SELECT * FROM supplier;
Output : output of the above command - SELECT NAME AS 'NAME_OF_SUPPLIER', ADDRESS AS 'ADDRESS_OF_SUPPLIER' FROM supplier; 
+---------------------+-----------------------+
| NAME_OF_SUPPLIER    | ADDRESS_OF_SUPPLIER   |
+---------------------+-----------------------+
| zoe                 | address_of_abhinav    |
+---------------------+-----------------------+
| backus              | address_of_backus     |
+---------------------+-----------------------+
But it does not change anything in the original table.

  • INSERT()

    SELECT INSERT('The invisible man',1,3,'wow'); TIPS → It has two arguments, the first argument tells about from where do you want to insert new word or new string and second argument denotes that the total length of that string which will be replace and display result .
    In this case we want to replace word 'The' of length 3, so the value of second argument is 3.

    Output : 
    +---------------------------------------+
| INSERT('The invisible man',1,3,'wow') |
+---------------------------------------+
| wow invisible man                     |
+---------------------------------------+
    If you don't want to replace anything from string 'The invisible man' then the query will be - SELECT INSERT('The invisible man',1,0,'wow '); TIPS → Here the second argument has value 0 and it means that we want to replace the string of length 0 at index 1 (value of first argument) from string 'The invisible man', but there is no any letter of length 0 at index 1 in string 'The invisible man' .

    Output : 
    +----------------------------------------+
| INSERT('The invisible man',1,0,'wow ') |
+----------------------------------------+
| wow The invisible man                  |
+----------------------------------------+
